# BeDelightful

BeDelightful is the flagship of the Delightful product family—a general-purpose AGI system built for complex task scenarios. With multi-agent design and a rich toolset, it supports autonomous task understanding, planning, execution, and self-correction. BeDelightful can follow natural language instructions, execute business workflows, and deliver end results.

As the flagship product, BeDelightful is open source to enable rapid customization. Teams can quickly build and deploy domain-specific assistants, boosting decision speed and quality.

## Key Features

- **Tool-rich**: Preloaded utilities for Python execution, web search, browser automation, file operations, and more
- **State management**: Robust lifecycle states including IDLE, RUNNING, FINISHED, and ERROR
- **Execution controls**: Limits for maximum iterations and output length
- **Multi-agent collaboration**: Coordinate multiple agents for complex tasks
- **Seamless integration**: Works with the Delightful Flow ecosystem for stronger workflow orchestration

## Requirements

- Python 3.12+
- LLM API keys (Claude, OpenAI, DeepSeek V3, Qwen)
- Project dependencies

## Environment Variables

Before running, set environment variables via a `.env` file or your system environment.

Required variables:
- `OPENAI_API_KEY`: OpenAI API key
